# DeBook
This is a used book marketplace built with PHP, JavaScript, and HTML, powered by a MySQL database.

## Deploying DeBook
- PHP 7.0 or above
- MariaDB or MySQL
- Any HTTP Server (Apache Httpd and Nginx are recommended)
