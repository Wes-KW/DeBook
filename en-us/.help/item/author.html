Add or Edit an Author Entry

Learn how to add or edit an author entry in DeBook.

Add an Author Entry

To add an author: Click the "Add or Edit: Author" button at the bottom of any page, or use any of the following ways to enter the author add/edit page:

  • Click the "Add Author" button on the author's homepage
  • Click "Add Author" on the author's detail page
  • Click "Enter Author" on the search page

Edit an Author Entry

To edit an author: Go to the author's information page and click the "Edit Author Introduction" button below the booklist introduction.

On the Author Information Edit Page:

  1. Add the author's picture (optional)
  2. Enter the author's Chinese name in the name input box
  3. Add the author's introduction
  4. Enter the Chinese names of the books authored in the works input box. Note:
    • Books already entered will appear as you type. You can click or use the up/down keys to select and add books, or manually enter books not already in the system.
    • Different books should be separated by an English semicolon (";")
    • Only books recognized by the input box can be used as search filters
  5. Add the author's search index; use the search index to add other language names or features for the author
  6. Click the "Continue" button

When Asked for Email:

By default, we will notify you of the review progress of your author edits via email. If you are not logged in, we will ask for your email address. If you do not wish to receive any related information, please leave the input box blank and click OK.
