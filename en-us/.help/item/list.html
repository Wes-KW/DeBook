Add or Edit a Booklist

Learn how to add or edit a booklist in DeBook.

Add a Booklist

To add a booklist: Click the "Add or Edit: Booklist" button at the bottom of any page, or use any of the following ways to enter the booklist add/edit page:

  • Click the "Add New Booklist" button on the booklist homepage
  • Click "Add Booklist" on the booklist detail page
  • Click "Create Booklist" on the search page

Edit a Booklist

To edit a booklist: Go to the original booklist information page, log in with the account you used to add the booklist, and click the "Edit Booklist Introduction" button below the booklist introduction.

On the Booklist Information Edit Page:

  1. Add a picture for the booklist (optional)
  2. Enter the Chinese name of the booklist in the name input box
  3. Enter tags for the booklist in the tag input box. Note:
    • Common tags will appear as you type. You can click or use the up/down keys to select and add tags, or manually enter tags not already in the system.
    • Different tags should be separated by an English semicolon (";")
    • Only tags recognized by the input box can be used as search filters
  4. Add a brief introduction for the booklist
  5. Enter the Chinese names of the books included in the booklist in the books input box. Note:
    • Books already entered will appear as you type. You can click or use the up/down keys to select and add books, or manually enter books not already in the system.
    • Different books should be separated by an English semicolon (";")
    • Only books recognized by the input box can be used as search filters
  6. Add the booklist's search index; use the search index to add other language names or features for the booklist
  7. Click the "Continue" button

When Asked for Email:

By default, we will notify you of the review progress of your booklist edits via email. If you are not logged in, we will ask for your email address. If you do not wish to receive any related information, please leave the input box blank and click OK.

  • Note: If you are not logged in when adding a booklist, you will not have permission to modify this booklist in the future.
