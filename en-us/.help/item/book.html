Add or Edit a Book

Learn how to add or edit a book in DeBook.

Add a Book

To add a book: Click the "Add or Edit: Book" button at the bottom of any page, or use any of the following ways to enter the book add/edit page:

  • Click the "Add Book" button on the DeBook homepage or the book homepage
  • Click "Add Book" on the book detail page
  • Click "Add Book" on the search page

Edit a Book

To edit a book: Go to the book's information page and click the "Edit Book Introduction" button below the book introduction.

On the Book Information Edit Page:

  1. Add at least one cover image of the book. It can be a cover of any version or language.
  2. Enter the book's name in the name input box
  3. Enter the book's tags in the tag input box. Note:
    • Common tags will appear as you type. You can click or use the up/down keys to select and add tags, or manually enter tags not already in the system.
    • Different tags should be separated by a semicolon (";")
    • Only tags recognized by the input box can be used as search filters
  4. Enter the author's name in the author input box. Note:
    • Authors already entered will appear as you type. You can click or use the up/down keys to select and add authors, or manually enter authors not already in the system.
    • Different authors should be separated by a semicolon (";")
    • Only authors recognized by the input box can be used as search filters and as links to the author's information page
  5. Add a brief introduction for the book
  6. Add the book's search index; use the search index to add other language names or features for the book
  7. Click the "Continue" button

When Asked for Email:

By default, we will notify you of the review progress of your book edits via email. If you are not logged in, we will ask for your email address. If you do not wish to receive any related information, please leave the input box blank and click OK.
